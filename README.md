# 🎫 Ticket Management Dashboard

A modern, scalable **Ticket Management Dashboard** built with **React, Redux, and Tailwind CSS**.
This application helps teams efficiently manage, track, and resolve tickets with a clean UI and optimized performance.

---

## 🚀 Features

* 🔐 Authentication (Login / Logout)
* 📊 Dashboard with key metrics
* 🎫 Ticket creation & management
* 🔍 Search and filter tickets
* 🌙 Dark / Light mode toggle
* ⚡ Optimized performance with centralized state
* 📱 Fully responsive design

---

## 🧠 Tech Stack

* **Frontend:** React.js
* **State Management:** Redux Toolkit
* **Styling:** Tailwind CSS
* **Routing:** React Router
* **Build Tool:** Vite / CRA (based on your setup)

---

## 📂 Folder Structure

```
src/
│── components/      # Reusable UI components
│── pages/           # Application pages (Dashboard, Tickets, etc.)
│── store/           # Redux store & slices
│── hooks/           # Custom hooks (e.g., useTheme)
│── services/        # API calls
│── assets/          # Images & static files
│── App.jsx
│── main.jsx
```

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/ticket-management-dashboard.git
cd ticket-management-dashboard
```

2. Install dependencies

```bash
npm install
```

3. Run the project

```bash
npm run dev
```
